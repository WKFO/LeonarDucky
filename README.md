# LeonarDucky
Rubber Ducky Payload Converter &amp; Generator for Arduino
# Usage
```
leonarducky.py -i <inputfile> -d <delay> -f <True/False> -o <outputfile>
-or-
leonarduckyGUI4.py - leonarduckyGUI5.py (Requires QT)
 ```
